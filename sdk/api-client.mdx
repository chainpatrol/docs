## Usage

```ts
import { ChainPatrol } from "@chainpatrol/sdk";

const chainpatrol = new ChainPatrol({ apiKey: "<API_KEY>" });
```

## API

### `new ChainPatrol({ apiKey: string })`

Creates a new instance of the ChainPatrol SDK.

#### Parameters

| Name   | Type   | Description                                                                  |
| ------ | ------ | ---------------------------------------------------------------------------- |
| apiKey | string | Your ChainPatrol API key. You can get one by creating an account on our site |

#### Returns

`ChainPatrol`

### `chainpatrol.asset.check()`

Checks if an asset is valid.

#### Parameters

| Name  | Type   | Description                                                                  |
| ----- | ------ | ---------------------------------------------------------------------------- |
| asset | string | The asset to check. Can be a ticker, contract address, or token ID (ERC-721) |

#### Returns

`Promise<AssetCheckResponse>`

### `chainpatrol.asset.list()`

Lists all assets that are currently being monitored.

#### Parameters

None

#### Returns

`Promise<AssetListResponse>`
