---
title: Page
---

Fake or scam page links that have been submitted from users and organizations.

When you sumbit a report, if you select asset type as `Page`, the url string won't be normalized and its original string will be submitted to **Chainpatrol**.

#### Available page domains

```
bit.ly
blur.io
goo.gl
link3.to
linktr.ee
opensea.io
t.co
t.me
telegram.me
tinyurl.com
twitter.com
beacons.ai
```
