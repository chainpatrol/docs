---
title: URL
---

All of scam site URL domains(includes their sub-domains) that have been submitted from users and organizations.

When you sumbit a report, if you select asset type as `URL`, the url string will be normalized and its domain will be submitted to **Chainpatrol**.

#### Report

```
https://www.chainpatrol.xyz/home
```

#### Result

```
chainpatrol.xyz
*.chainpatrol.xyz
*.*.chainpatrol.xyz
...
```

<Warning>
  It shouldn't be such like twitter or telegram page links. `Page` glossary is
  described about them.
</Warning>
